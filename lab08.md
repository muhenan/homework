# 游戏开发

## 一、游戏策划

>背景：
>小米是一个足球踢得很差的同学，他很难进球，所以他想踢中球门就得分，
>并且，击毁球门
>
>
>玩法：
小米通过 wasd 控制球的移动，让球击中球门，然后球门消失  
>对象：1.球：被wasd控制移动
>
>2.球门：被球击中后消失

### 1.
+---------------------------------+    
| 球                    |    
+---------------------------------+    
| heighth, length, width; place                         |    
+--------------+------------------+   
| 球门 | 无 |   
+--------------+------------------+   
|              |                  |    
+--------------+------------------+    

### 2.

+---------------------------------+    
| 球门                     |    
+---------------------------------+   
| heighth, length, width; place                      |   
+--------------+------------------+   
| 球 | 被球击中时消失 |   
+--------------+------------------+   
|              |                  |    
+--------------+------------------+
